Color

A near-black, an ivory, and a single signal.

04 / 11
Buttons

Sharp, flat, declarative.

2px radius — not rounded. Inter 500, uppercase, 14% tracking. Action verbs only — Request, Explore, Talk to dispatch. The arrow nudges 4px on hover.

PRIMARY

Clearance fill, ink text. Hover lifts to clearance-hover; never grows.

SECONDARY

1px bone border, transparent fill. Hover fills to 8% bone alpha.

GHOST

Text only, bone-2 default. Underline animates in on hover, 1.5px clearance.
